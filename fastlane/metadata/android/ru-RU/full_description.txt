Минимальное приложения для изменения состояния Wi-Fi через отправку широковещательных сообщений.
Полезно для обхода ограничения компании Google на приложения использующие API 29 и выше.

После включения, чтобы изменить состояние Wi-Fi, Вы должны отправить широковещательное сообщение
на "me.lucky.wyfy/.ControlReceiver" с одним из действий внизу. Также Вы можете требовать
аутентификацию по коду, чтобы ограничить доступ к контроллеру, сделав долгий клик на блоке с кодом
(красный - выключена, желтый - включена). Пока включена аутентификация, Вы должны отправлять этот
код с каждым широковещательным сообщением, в дополнении с переменной "code".

Широковещательные сообщения можно отправлять в приложении Key Mapper, в приложениях для
автоматизации, таких как Automate/Tasker/MacroDroid и так далее.

Действия:
* me.lucky.wyfy.action.SET_ON
* me.lucky.wyfy.action.SET_OFF

Разрешения:
* CHANGE_WIFI_STATE - изменения состояния Wi-Fi

Это свободное программное обеспечение с открытым исходным кодом.
